Markdown Syntax

Markdown 是一個輕量級標記式語言(wiki)，它是以純文字的方式編寫文件， 使用起來很方便，我喜歡用它來做筆記，它還可以表示數學表示式(Latex語法)且能夠輕易地將其轉成其他格式(HTML, PDF) 以下針對幾個常用的語法做記錄：
